## DMIT 支持哪些付款方式？

先给个总览：DMIT 目前支持以下付款渠道——

- **支付宝（Alipay）**
- **微信支付（WeChat Pay）**
- **PayPal**
- **信用卡 / 借记卡**（通过 Stripe 处理）
- **银行转账**
- **加密货币**（USDC、USDP、USDG，支持 Ethereum、Polygon、Base 链）

对国内用户来说，支付宝和微信是最直接的选择。境外用户或者不想绑国内账户的，PayPal 和信用卡都没问题。加密货币是 DMIT 后来新加的选项，走稳定币，不用担心汇率波动。

## 各付款方式详解

### 支付宝 —— 国内用户首选

扫码或者跳转 APP 完成支付，整个流程跟国内买东西没什么两样。即时到账，付完钱几分钟内 VPS 就开始创建了。

没有额外手续费（支付宝本身不收，DMIT 也不加收）。唯一要注意的是，结算是美元，支付宝会按当时汇率换算，实际扣的是人民币。汇率就是当天的市场价，不存在什么特别贵或者特别坑的问题。

### 微信支付 —— 手机端更顺手

跟支付宝逻辑一样，扫码或者跳微信 APP 完成支付。到账速度也是即时的。

如果你手机上微信比支付宝用得更顺，选这个就行，没区别。

### PayPal —— 境外账户 / 不想绑国内卡的首选

PayPal 需要提前有账户余额或者绑卡。跳转到 PayPal 完成支付后再回来，流程正常情况下两三分钟搞定。

有一点：PayPal 本身可能收货币转换费，如果你的 PayPal 账户余额是人民币，转美元时会有一点损耗。建议账户里提前备好美元余额，或者直接绑一张美元信用卡走 PayPal。

### 信用卡（Stripe）—— 国际通用，无需账户

Visa、Mastercard、美国运通都行。通过 Stripe 处理，页面直接填卡号，不需要额外注册任何账户。

国内双币信用卡也能用，但偶尔会遇到银行风控拦截的情况。如果刷不过，换支付宝或 PayPal 就解决了。

### 银行转账 —— 适合大额或企业付款

转账到指定账户。这种方式到账时间相对慢，通常需要等待人工确认。日常个人购买不推荐，走支付宝或 PayPal 更快。

如果是公司采购、金额比较大，或者需要对公转账留记录的场景，可以选这个。

### 加密货币 —— 稳定币，支持退款

这是 DMIT 2025 年底新增的付款方式。支持 USDC、USDP、USDG 三种稳定币，链上选 Ethereum、Polygon 或者 Base。

退款在加密货币支付下也可以操作。不过要注意：付款时的网络手续费（Gas Fee）由支付提供商收取，退款时不会把这部分退回来，付款金额本身会退。

适合本来就持有稳定币、或者不想走传统支付渠道的用户。

## 常见问题

**Q：国内用户直接用支付宝付美元没问题吗？**

A：完全没问题。支付宝会自动按当天汇率换算，直接扣人民币。不需要提前换汇，也不需要美元账户。

**Q：信用卡付款失败怎么办？**

A：国内银行的双币卡偶尔会被风控拦截。换支付宝或微信付款就行，到账同样即时。如果坚持用卡，可以先联系发卡银行确认是否开通了境外消费权限。

**Q：DMIT 付款方式里看到的"银行转账"是什么？**

A：就是传统电汇，打款到 DMIT 指定银行账户，等人工确认后开通服务。速度慢，手续麻烦，适合企业对公场景。个人用户没必要走这个，支付宝两三分钟搞定。

**Q：加密货币付款安全吗，退款真的能退回来？**

A：DMIT 官方说明退款在加密货币下可操作，退款的是你付出去的稳定币金额。注意链上 Gas 手续费是支付网关收的，那部分不退。付款前确认好链的选择，Gas 费在 Base 链上通常比 Ethereum 便宜很多。

**Q：DMIT 目前有有效优惠码吗？**

A：截至当前，DMIT 没有公开的通用长期优惠码在跑。偶尔会针对特定机房或特定套餐推出阶段性折扣，建议直接进选购页面看当前价格是否已包含促销。
